A *mock* version of the ***Pokémon*** series, more specific, the *Fire Red*/*Leaf Green* games from *Gameboy Advance*.  

## Controls 🎮

- **Arrows** - Control the character and menus
- **Enter** - Selection
- **S** - Open up *pokémon* information menu
